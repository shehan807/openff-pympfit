Getting Started
===============

Installation
------------

**Prerequisites:**

- Python 3.9+
- Psi4 (for GDMA calculations)
- openff-recharge

**Install from source:**

.. code-block:: bash

    git clone https://github.com/shehan807/pympfit.git
    cd pympfit
    pip install -e .

**Note:** GDMA functionality requires Psi4 and PyGDMA:

.. code-block:: bash

    pip install psi4 pygdma
